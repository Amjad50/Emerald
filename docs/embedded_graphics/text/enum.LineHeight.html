Enum embedded_graphics::text::LineHeight

source · 
pub enum LineHeight {
    Pixels(u32),
    Percent(u32),
}
Expand description

Text line height.

The line height is defined as the vertical distance between the baseline of two adjacent lines of text.

Variants§

§

Pixels(u32)

Absolute line height in pixels.

§

Percent(u32)

Relative line height in percent of the default line height.

source

pub const fn to_absolute(self, base_line_height: u32) -> u32

Converts the line height to an absolute pixel distance.

Examples
use embedded_graphics::text::LineHeight;

let relative_height = LineHeight::Percent(150);
assert_eq!(relative_height.to_absolute(20), 30);
